## Wallet Events
Reasoning trace may include wallet-related events, such as signature requests, identity checks, and connection status changes. These events are logged for audit and debugging.

## Metadata
Reasoning trace includes metadata for each cognitive step, handler activation, persona shaping, and Web3 event. Metadata is used for audit, UI display, and backend integration.
